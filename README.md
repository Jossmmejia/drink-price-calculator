# Drink Price Calculator

## Description

This project is a simple Python program.  
It is a small system that calculates the total price of a drink order.

The program is based on a list of products with their prices.  
It is designed to practice loops, conditions, and user input in Python.

## How it works

1. The program creates a list of drinks with their prices.
2. The system asks the user what drink they want.
3. The program asks the quantity of drinks.
4. The program searches for the drink in the list.
5. The system calculates the total price.
6. The program prints the final result.

## Status

> The program is currently receiving user input and calculating the total price of the selected drink.
